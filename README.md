# ESP32-Desk-Companion

The Desk Companion is an ESP32-based embedded systems project designed to improve desk productivity through physical controls and visual feedback. Rather than relying solely on software, the device provides dedicated hardware controls for timers, alarms, study modes, and future smart features.

## Why I'm Building It

This project is intended to strengthen my embedded systems skills by combining electronics, firmware development, and hardware design into a single practical device which I use on a daily basis. It also serves as a portfolio project demonstrating my ability to design, build, and document a complete embedded system.

## Planned Features
- RGB status modes
- Adjustable brightness
- Timer
- Alarm clock
- Stop Watch
- Productivity reminders
- Physical controls
- Web dashboard
- Presence detection
